# Hindi Pronouncer Chrome Extension

A simple Chrome extension to help you learn the Hindi pronunciation and meaning of English words right from your browser.

## Features

*   **Text-to-Speech:** Hears the American English pronunciation of any word you select.
*   **Hindi Transliteration:** Get the phonetic pronunciation of the English word written in Hindi script.
*   **Hindi Meaning:** Instantly find the Hindi translation of the selected word.
*   **Seamless Integration:** Works on any webpage using a simple right-click context menu.
*   **Sleek Pop-up:** Displays the information in a clean, non-intrusive pop-up that disappears automatically.

## How to Use

1.  **Select a word:** Highlight any English word on a webpage.
2.  **Right-click:** Open the context menu.
3.  **Choose the option:** Click on "Pronounce & Hindi Meaning".
4.  **Learn:** The word will be spoken out loud, and a pop-up will appear in the top-right corner of the page with its pronunciation and meaning in Hindi. The pop-up will close automatically after 8 seconds, or you can click it to dismiss it sooner.

## Installation

You can install this extension locally by following these steps:

1.  **Download:** Download or clone this repository to your local machine.
2.  **Open Chrome Extensions:** Open the Google Chrome browser and navigate to `chrome://extensions`.
3.  **Enable Developer Mode:** Turn on the "Developer mode" toggle, usually found in the top-right corner.
4.  **Load the Extension:** Click on the "Load unpacked" button and select the directory where you saved this extension's files.

The extension is now installed and ready to use!
